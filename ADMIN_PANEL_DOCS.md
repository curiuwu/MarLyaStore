# 📋 Документация Защищённой Админ-Панели

Полное руководство по использованию админ-панели MarLya Store.

---

## 🚀 Быстрый старт

### 1. Создание администратора через CLI

Команда для создания администратора:

```bash
flask create-admin
```

Программа попросит следующие данные:

```
Email администратора: admin@example.com
Пароль (минимум 6 символов): SecurePassword123
Имя: Admin
Фамилия: Иванов
```

После успешного создания вы получите подтверждение:

```
✅ Администратор 'Admin Иванов' (admin@example.com) успешно создан
📧 Email: admin@example.com
```

### 2. Вход в админ-панель

1. Перейдите на страницу входа: `http://localhost:8080/auth/login`
2. Введите email и пароль администратора
3. После входа перейдите в свой профиль
4. Нажмите на ссылку "👑 Админ-панель"

Или прямая ссылка: `http://localhost:8080/admin/`

---

## 🛡️ Безопасность

### Защита доступа

- **Только администраторы** имеют доступ к админ-панели
- Если неавторизованный пользователь попытается войти → перенаправление на login
- Если авторизованный, но не админ попытается войти → ошибка 403
- Все действия логируются в файл `logs/app.log`

### Требования к паролю

- Минимум 6 символов
- Пароль хешируется с помощью `werkzeug.security` (алгоритм scrypt)
- Пароли **никогда не хранятся в открытом виде**

## 📦 CRUD Товаров

### Просмотр списка товаров

**URL:** `/admin/products`

Возможности:
- 📋 Все товары с пагинацией (20 на странице)
- 🔍 Поиск по названию
- 🏷️ Фильтр по категориям
- 📄 Пагинация

### Создание товара

**URL:** `/admin/products/create`

Обязательные поля:
- **Название** (минимум 3 символа)
- **Описание** (минимум 10 символов)
- **Категория** (выбрать из списка)

Опциональные поля:
- URL изображения

**Скриншот процесса:**

```
1. Нажимаешь кнопку "➕ Создать товар"
2. Заполняешь форму
3. Нажимаешь "✅ Создать товар"
4. Получаешь успешное уведомление
5. Перенаправление на список товаров
```

### Редактирование товара

**URL:** `/admin/products/<PRODUCT_ID>/edit`

Можешь редактировать:
- Название
- Описание
- Категорию

### Удаление товара

**URL:** `/admin/products/<PRODUCT_ID>/delete`

⚠️ **Важно:**

- Удаление требует подтверждения
- Вместе с товаром удаляются:
  - Все изображения товара
  - Все скидки
  - Все отзывы/рецензии

**Безопасно:**
- Нет случайного удаления по GET-запросу
- POST метод с явным подтверждением

### Файлы реализации

| Файл | Назначение |
|------|-----------|
| `app/helpers.py` | Декораторы для защиты (`@admin_required`) |
| `app/routes/admin_routes.py` | Routes для админ-панели (CRUD товаров) |
| `app/templates/admin/base.html` | Базовый layout администратора |
| `app/templates/admin/dashboard.html` | Главная страница админки |
| `app/templates/admin/products/list.html` | Список товаров |
| `app/templates/admin/products/create.html` | Форма создания |
| `app/templates/admin/products/edit.html` | Форма редактирования |
| `app/templates/admin/products/delete.html` | Подтверждение удаления |

### Декоратор @admin_required

```python
from app.helpers import admin_required

@admin_bp.route('/admin-only')
@admin_required
def admin_only_view():
    return "Only admins can see this"
```

Что делает:
1. Проверяет, авторизован ли пользователь
2. Проверяет, есть ли у него роль `admin`
3. Логирует все попытки несанкционированного доступа
4. Перенаправляет или возвращает 403

## 🔄 Работа с товарами через API

### Получить список товаров (JSON)

```bash

```

Параметры:
- `search` - поиск по названию
- `category` - фильтр по категории ID
- `page` - номер страницы

### Удалить товар через API

```bash

```

Ответ:
```json
{
  "success": true,
  "message": "Товар \"Название\" успешно удалён"
}
```

## 🐛 Торубление

### Проблема: "У вас нет доступа к админ-панели"

**Решение:** Убедитесь, что:
- Вы авторизованы
- Ваша роль `admin` (созданы через `flask create-admin`)

Проверить роль:
```python
# В профиле можно увидеть роль
current_user.role.role_name
```

### Проблема: Ошибка 404 при попытке входа

**Решение:** Проверьте, что вы используете правильный URL. Должно быть:
```
http://localhost:8080/admin/
```

Не: `/admin` (без слеша в конце)
